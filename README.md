> 대사관 외부 인사 컨택트 포인트 통합 관리 프로그램
> 

---

## 1. 프로젝트 개요

대사관에서 외부 인사를 관리하는 역할을 맡았으나, 관련 정보가 여러 곳에 흩어져 있어 컨택트 포인트를 찾는 데 시간이 과도하게 소요되는 문제가 발생함.

정보를 찾는 데만 시간을 쓰다 보니 정작 중요한 커뮤니케이션과 본질적인 업무에 집중하기 어려운 상황이 반복됨. 이러한 비효율을 해소하고자 외부 인사 정보를 한 곳에서 관리하고 즉시 조회할 수 있는 인적 자원 관리 시스템을 개발하게 됨.

| 항목 | 내용 |
| --- | --- |
| 개발 배경 | 대사관 외부 인사 정보 분산으로 인한 업무 비효율 해소 |
| 핵심 문제 | 컨택트 포인트 탐색 시간 과다 → 실제 커뮤니케이션 지연 |
| 해결 방향 | 인사 정보 통합 관리 및 즉시 검색 가능한 시스템 구축 |

---

## 2. 해결하려는 문제

**🔍 외부 인사를 빠르게 검색할 수 없을까?**
이름, 소속 기관, 분야 등 다양한 조건으로 외부 인사를 즉시 검색할 수 있는 기능이 필요했음. 기존에는 엑셀이나 메모 형태로 분산 관리되어 원하는 인사를 찾는 데 시간이 걸렸음.

**🏢 기관별 인사를 한눈에 볼 수 없을까?**
같은 기관 소속 인사들을 한 화면에서 묶어서 볼 수 있는 기관별 뷰가 필요했음. 어떤 기관과 얼마나 많은 인적 네트워크가 구축되어 있는지 파악이 어려웠음.

**🌐 분야별 교류 현황을 파악할 수 없을까?**
어떤 분야에서 어떤 인사와 교류가 있었는지 이력을 관리하고 시각적으로 파악할 수 있는 기능이 필요했음.

---

## 3. 주요 기능

### 기능 1 — 인명 기반 인사 검색

- 이름으로 외부 인사를 즉시 검색
- 동명이인 구분을 위한 소속 기관 및 직함 함께 표시
- 검색 결과에서 바로 상세 정보 확인 가능

### 기능 2 — 기관별 인사 분류

- 소속 기관에 따라 인사를 그룹화하여 표시
- 기관별 전체 컨택트 포인트 한눈에 조회
- 기관 내 담당자 변경 시 이력 관리
